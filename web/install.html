Installation Instructions:

On Phone:

visit .apk file URL on your phone

Using Computer and Android SDK:

  1. Install the Google USB Driver Package through the SDK Manager
  2. Download the .apk file
  3. Open Command Prompt (windows) / Terminal (mac/linux)
  4. cd to the platform-tools directory in the sdk files installed
  5. run: adb path/RecipeReader.apk
